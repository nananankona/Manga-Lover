# Manga-Lover

**Education Only**

## 概要

mangalove.me から漫画をダウンロード＆複合化できるツールです。

## How to Use

### 動作環境
- Node.js v18.x 以上
- インターネット接続環境
- Git
当たり前だよなぁ？

### インストール方法

1. Clone&cd:
```bash
git clone https://github.com/nananankona/Manga-Lover.git && cd Manga-Lover && npm install
```

### Run
```bash
node app.js
```
